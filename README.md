# 🔐 Caesar Cipher - Python Project

A simple Python project that encrypts and decrypts messages using the Caesar Cipher technique.

## 🚀 About

The Caesar Cipher is one of the oldest encryption techniques.  
It works by shifting letters in the alphabet by a fixed number.

Example:

```text
HELLO → KHOOR
```

(Shift = 3)
